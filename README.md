# 智慧校园信息数据大屏 (2023 毕业设计)

## 项目启动

### 后端 (WebSocket)

```
cd ./backend
cnpm i
node server.js
```

### 前端 (Vue)

```
cd ./frontend
cnpm i
cnpm run serve
```
